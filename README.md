# ⚽ UEFA Country Ranking — Live Tracker

A self-hosted, fully open-source clone of footballseeding.com's UEFA Country Coefficient tracker.

- 🟢 **Live updates** every 3 minutes on match nights via GitHub Actions
- 📊 **All seasons** from 2008-09 to present
- 🚀 **Free hosting** on Netlify or Vercel
- 🐍 **Python scraper** with UEFA.com as primary source, fallback to footballseeding

---

## 🚀 Deploy in 5 steps

### Step 1 — Create your GitHub repository

1. Go to https://github.com/new
2. Name it `uefa-ranking` (or anything you like)
3. Set it to **Public** (required for free GitHub Actions minutes)
4. Push this project:

```bash
git init
git add .
git commit -m "initial commit"
git remote add origin https://github.com/YOUR_USERNAME/uefa-ranking.git
git push -u origin main
```

### Step 2 — Run the scraper for the first time

You can trigger it manually from GitHub:
1. Go to your repo → **Actions** tab
2. Click **"Scrape UEFA Rankings"**
3. Click **"Run workflow"** → set season to `all`
4. Wait ~2 minutes

Or run it locally first:
```bash
cd scraper
pip install -r requirements.txt
python scraper.py --all     # scrape all seasons
python scraper.py           # scrape current season only
```

### Step 3 — Deploy on Netlify (recommended, free)

1. Go to https://app.netlify.com → "Add new site" → "Import an existing project"
2. Connect your GitHub account and select `uefa-ranking`
3. Settings:
   - **Build command**: *(leave empty)*
   - **Publish directory**: `public`
4. Click **Deploy**
5. Your site is live! Netlify gives you a URL like `https://uefa-ranking-abc123.netlify.app`

### Step 4 — Deploy on Vercel (alternative)

```bash
npm i -g vercel
vercel --prod
```
Or connect via https://vercel.com/new and import from GitHub.
Set **Output Directory** to `public`.

### Step 5 — Enable automatic updates

The GitHub Action runs automatically on schedule (every 3 minutes on match nights).
It will:
1. Run the scraper
2. Commit updated JSON files to your repo
3. Netlify/Vercel automatically redeploys (takes ~30 seconds)

No server needed. Everything runs on GitHub's free tier.

## 🔧 Customization

### Change the refresh interval

Edit `.github/workflows/scrape.yml` and modify the `cron` expressions.

Match nights (UEFA):
- **Champions League**: Tue + Wed
- **Europa League**: Thu
- **Conference League**: Thu

### Add your custom domain

On Netlify: Site settings → Domain management → Add custom domain

### Force-refresh a specific season

Go to Actions → "Scrape UEFA Rankings" → Run workflow → enter the season (e.g. `2024-25`)

## 📋 UEFA Coefficient rules

- Points: Win = 2, Draw = 1 (qualifying: Win = 1, Draw = 0.5)
- Bonus: Group stage CL = 4pts, R16 CL = 5pts, QF/SF/F = 1pt each
- Country coefficient = sum of club points ÷ number of clubs entered
- **Total = sum of last 5 annual coefficients**
